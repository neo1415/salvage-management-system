## Dimensions (A4 Page)

| Element | Position | Size |
|---------|----------|------|
| **Page** | - | 210mm × 297mm |
| **Letterhead Bar** | 0mm | 35mm tall |
| **Company Name** | 12mm from top | 20pt, bold, white |
| **Address** | 20mm from top | 9pt, normal, white |
| **Contact** | 27mm from top | 8pt, normal, white |
| **Report Title** (page 1 only) | 42mm from top | 14pt, bold, black, centered |
| **Generated Date** (page 1 only) | 48mm from top | 10pt, normal, gray, centered |
| **Content Start** (page 1) | 54mm from top | - |
| **Content Start** (page 2+) | 40mm from top | - |
| **Footer** | 287mm from top (10mm from bottom) | 8pt, gray |
| **Left Margin** | 10mm | - |
| **Right Margin** | 10mm | - |
