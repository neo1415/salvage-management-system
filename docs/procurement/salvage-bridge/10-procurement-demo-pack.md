# Procurement Demo Pack

Use this as the recommended minimum set before serious insurer procurement conversations.

## First Meeting Pack

Send only:

1. Product overview
2. Security overview
3. White-label model summary
4. Access control matrix
5. Subprocessor draft list
6. Support/SLA overview

Avoid sending:

- raw code
- environment variable lists with secrets
- internal troubleshooting documents
- unreviewed legal templates
- detailed database schema unless requested by technical security review

## Technical Review Pack

Add:

1. Architecture and deployment overview
2. Data flow summary
3. Access control matrix
4. Security overview
5. Backup/recovery draft
6. Incident response draft
7. Vulnerability management draft

## Legal / Compliance Review Pack

Add:

1. DPA draft
2. Privacy notice template
3. Data retention policy draft
4. Subprocessor list
5. NDPA/NDPC readiness notes
6. Provider compliance references

## Executive Summary Language

> Salvage Bridge provides a configurable white-label salvage recovery platform for insurers, designed to support verified vendor onboarding, salvage auctions, wallet/deposit handling, document signing, payment tracking, audit logs, and operational reporting. Formal compliance certifications and client-specific legal terms are handled through the applicable audit, legal, and procurement process.
