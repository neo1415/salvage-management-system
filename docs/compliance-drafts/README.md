## Drafting Position

The documents use practical, non-overclaiming language:

- "designed to"
- "intended to"
- "where configured"
- "as agreed in the applicable order form"
- "subject to provider capability"
- "client-specific deployment"

Avoid converting these into absolute guarantees unless the operational process, monitoring, insurance, legal terms, and provider commitments support the guarantee.
